Functional Documentation

Overview

This section of the documentation provides end-user, platform-independent documentation of QuantLibAddin/QuantLibXL functionality.

Section Manual summarizes selected topics concerning end-user functionality of ObjectHandler, QuantLibAddin, and QuantLibXL.

Section Functions is an alphabetic list of links to the documentation for all functions in QuantLibAddin.

Section Categories is a list of links to documentation of QuantLibAddin functions by category.

Section Enumerations documents alphanumeric codes which QuantLibAddin recognizes to indicate minor or transient QuantLib classes which are constructed as part of larger operations.

Section Framework gives an overview of the QuantLibXL Framework, an application layer which wraps the QuantLibXL Addin with additional functionality in support of market data feeds, creation of instruments, etc.
