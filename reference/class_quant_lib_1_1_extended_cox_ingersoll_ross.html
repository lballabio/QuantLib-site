Detailed Description

Extended Cox-Ingersoll-Ross model class.

This class implements the extended Cox-Ingersoll-Ross model defined by

\[ dr_t = (\theta(t) - \alpha r_t)dt + \sqrt{r_t}\sigma dW_t . \]
