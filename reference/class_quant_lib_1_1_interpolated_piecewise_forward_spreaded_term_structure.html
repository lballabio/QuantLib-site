Detailed Description

template<class Interpolator>
class QuantLib::InterpolatedPiecewiseForwardSpreadedTermStructure< Interpolator >

Term structure with an added vector of spreads on the instantaneous forward rate.

The forward rate spread at any given date is interpolated between the input data.

Note
This term structure will remain linked to the original structure, i.e., any changes in the latter will be reflected in this structure as well.
