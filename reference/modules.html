 Financial instruments
 Finite-differences frameworkThis framework (corresponding to the ql/FiniteDifferences directory) contains basic building blocks for the numerical solution of partial differential equations by means of finite-difference methods
 Lattice methodsThe framework (corresponding to the ql/Lattices directory) contains basic building blocks for pricing instruments using lattice methods (trees). A lattice, i.e. an instance of the abstract class QuantLib::Lattice, relies on one or several trees (each one approximating a diffusion process) to price an instance of the DiscretizedAsset class. Trees are instances of classes derived from QuantLib::Tree, classes which define the branching between nodes and transition probabilities
