void ObjectHandler::Registry< KeyClass >::deleteTypeMap ( const std::string &  mapID) const
inline

Free any memory allocated to the type map.

If the derived class has called "new" when initializing the type map, then when the application shuts down, a (harmless) memory leak will occur. This function should be called to ensure a clean shutdown.

◆ getAllTypesMap()

template<typename KeyClass>
virtual AllTypeMap& ObjectHandler::Registry< KeyClass >::getAllTypesMap ( ) const
pure virtual

Return a reference to the type map.

The type map would normally be declared as a member variable but that isn't possible because std::map cannot be exported across DLL boundaries. Instead this pure virtual member function is declared in the base class. Derived classes must implement a concrete override of the function, returning a (non-const) reference to a static variable declared in a cpp file.
