Detailed Description

Capture the values of the arguments passed to the Object constructor.

For each class derived from Object there is a corresponding ValueObject class. The source code of classes derived from ValueObject is generated automatically by gensrc. Each Object holds a reference to its ValueObject.

The ValueObject captures a snapshot of the input arguments that were passed to the Object's constructor. These values can be used later for serialization or run-time interrogation of the Object.

This class's member functions are implemented inline in this header file. This is necessary to work around problems on the Windows platform when exporting this class across DLL boundaries.
