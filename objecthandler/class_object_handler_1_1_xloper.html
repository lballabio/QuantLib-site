Detailed Description

Perform RAII for Excel's XLOPER datatype.

This class implements a thin wrapper for an Excel XLOPER value. The class is intended specifically to manage XLOPERs which are declared on the stack and passed to Excel in order to receive the return value.

~Xloper calls xlFree on the underlying XLOPER, relieving the client code from the need to manage this e.g. in the event of the stack unwinding after an exception.
