Detailed Description

template<typename T>
class ObjectHandler::Create< T >

Create the enumerated type associated with a string.

Functor comprising a concrete instantiation of base template class RegistryManager. The () operator returns the required type, e.g.

Account::Type accountTypeEnumeration =
ObjectHandler::Create<Account::Type>()("Savings");
