ObjectHandler

Overview

ObjectHandler implements a repository in which objects may be instantiated, queried, updated, and destroyed, facilitating deployment of object oriented functionality to procedural platforms such as spreadsheets. Core functionality is implemented in platform neutral C++ and an interface to Microsoft Excel is supported.

QuantLib

ObjectHandler is a subproject of the QuantLib C++ analytics library and shares the QuantLib project structure with regard to distribution and licensing. The mailing list for ObjectHandler is quantlib-users@lists.sf.net. The list archives may be searched here.
