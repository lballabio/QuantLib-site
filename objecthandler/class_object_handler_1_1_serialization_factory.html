Detailed Description

A Singleton wrapping the boost::serialization interface.

The pure virtual functions in this class must be implemented as appropriate for client applications.

Recreate an Object from its ValueObject and store it in the Repository.

This function calls recreateObject to recreate the Object from its ValueObject then stores the newly created Object in the Repository with a call to Repository::storeObject().
